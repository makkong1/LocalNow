# PRD: LocalNow

## 목표
여행자가 "지금 이 순간" 겪는 정보·언어·긴급 문제를, 주변의 현지 가이드와 실시간으로 매칭해 즉시 해결한다.

## 사용자
- **여행자 (Traveler)**: 20~30대 자유여행객, 디지털 노마드, 장기 해외 체류자, 출장 방문자.
- **현지 가이드 (Guide)**: 해당 도시에 거주하며 특정 언어·관광·통역 역량을 가진 검증 사용자.
- **운영자 (Admin)**: 가이드 인증, 신고/환불 처리. MVP 에서는 DB 직접 접근으로 갈음한다.

## 핵심 기능 (MVP 포함)

### 서버 기능
1. **계정 & 인증**: 이메일/비밀번호 회원가입, 로그인, JWT 발급/갱신. 프로필에 사용 언어 / 거주 도시 / 역할(Traveler|Guide) 저장.
2. **도움 요청 생성**: 여행자가 현재 위치(lat, lng)와 요청 유형(GUIDE | TRANSLATION | FOOD | EMERGENCY 등), 시작 시각, 소요 시간, 제안 금액을 포함한 요청을 생성한다. 상태 머신: `OPEN → MATCHED → IN_PROGRESS → COMPLETED | CANCELLED`.
3. **주변 가이드 매칭**: Redis GEO 로 반경 N km 안의 가용 가이드를 조회하고 가이드 측에 알림 큐로 푸시. 가이드가 `accept` → 여행자가 후보 중 1명을 `confirm` → `MATCHED` 로 전환. 중복 확정은 분산락 + 낙관적 락으로 차단.
4. **실시간 채팅**: 매칭 확정된 요청 단위로 STOMP over WebSocket 채널을 연다. 메시지는 DB 영속화, 오프라인 상대에게는 RabbitMQ 알림 이벤트 발행.
5. **결제 흐름 (Mock PG)**: 요청 확정 시 결제 의도 → 완료 시 캡처 → 플랫폼 수수료(기본 15%, EMERGENCY 25%) 계산 및 분배 기록. 외부 PG 는 Mock.
6. **가이드 평점/리뷰**: 요청이 `COMPLETED` 가 되면 여행자가 별점(1~5) + 리뷰 작성. 가이드 프로필에 평균 평점 집계.

### 웹 화면 (데모 클라이언트)
목표는 "한 브라우저의 두 탭 = 여행자 + 가이드" 로 매칭과 채팅이 실시간으로 움직이는 걸 보여주는 것. 운영 수준 UI 가 아니다.

- **공통**: 로그인 / 회원가입 페이지, 로그아웃, 상단에 현재 사용자 / 역할 / 연결 상태 표시.
- **여행자 뷰** (`/traveler`)
  - 지도 위에서 현재 위치 기반으로 도움 요청 생성 폼(유형, 시간, 제안 금액).
  - 내 요청의 현재 상태 패널 (`OPEN` → 수락한 가이드 후보 목록 → 확정 → `MATCHED`).
  - 가이드 후보 카드에서 평점/리뷰 확인 후 "이 가이드로 확정" 버튼.
  - 확정 이후 채팅 패널 활성화, Mock 결제 버튼.
- **가이드 뷰** (`/guide`)
  - "가용(on-duty) 토글" — 켜면 주변 요청이 실시간으로 목록에 꽂힘.
  - 요청 카드에서 거리 / 유형 / 보상 확인 후 "수락" 버튼.
  - 여행자가 나를 확정하면 채팅 패널 활성화.
- **공통 채팅 패널**: 매칭된 요청 단위 방. 메시지 시간/발신자 표시. STOMP 연결 상태 인디케이터.

## MVP 제외 사항
- 모바일 앱(iOS/Android) 구현. 데모 웹으로 시연을 대체한다.
- 실시간 채팅 번역 API 연동. `Translator` 인터페이스에 pass-through Fake 만 제공.
- 긴급 상황 AI 챗봇. 요청 유형 `EMERGENCY` 는 일반 매칭 흐름으로 처리.
- 실제 PG 연동. Mock 만 제공.
- 추천 시스템(일정/취향 기반 자동 추천).
- 광고 제휴, 여행 보험 제휴.
- 다국어 i18n. 한국어/영어 문자열만 하드코딩.
- (예외) **관리**: 전사 Admin 제품은 제외. 데모 웹에 `UserRole.ADMIN` 전용 **읽기** 대시보드(집계)만 둔다. 상세는 **ADR-014**. 그 밖의 운영 쓰기(신고/환불 승인 등)는 이 범위 밖.
- 본격적인 반응형 / 접근성 튜닝. 데스크톱 1280px 기준으로만 맞춘다.

## 디자인 방향
- **다크 고정, 미니멀, 도구형 대시보드**. 랜딩 페이지가 아니라 매일 쓰는 콘솔 느낌.
- 포인트 컬러 **1가지만 사용**: amber/orange 계열(`#f59e0b`). 보라/인디고 금지.
- 지도는 Leaflet + OpenStreetMap 타일. Google Maps / Mapbox API 키 사용 금지.
- 상세 스타일 규칙과 금지 패턴은 `docs/UI_GUIDE.md` 단일 기준.
